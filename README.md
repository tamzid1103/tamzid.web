# Tamzidul Haque Portfolio Website

This repository contains a responsive personal portfolio website built with plain HTML, CSS, and a small amount of vanilla JavaScript.

## Pages

- Home: introduction, skills, experience, and education
- Projects: structured placeholder section ready for real project case studies
- Contact: accessible contact form and booking link

## Key Improvements Implemented

- Consistent design system across all pages using one shared stylesheet
- Semantic page structure with header, nav, main, sections, and footer
- Mobile navigation with accessible toggle behavior
- Skip-to-content link and stronger keyboard focus visibility
- Reduced-motion support for motion-sensitive users
- Safer external links with rel attributes
- SEO/social metadata on all pages (Open Graph and Twitter tags)
- JSON-LD Person structured data on the homepage

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Optional particles background effect on the homepage

## How To Add Projects Later

Edit projects.html and replace each placeholder card with real content:

1. Project title
2. Problem statement
3. Your solution and architecture
4. Technologies used
5. GitHub repository link
6. Live demo or screenshots
7. Measurable result or key learning

## Run Locally

Because this is a static website, open index.html directly in a browser.

For a cleaner local server workflow, you can use any static server, for example:

```bash
python -m http.server 5500
```

Then visit:

http://localhost:5500
